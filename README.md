# ToDo list application: 

In this application user can: 

1- Add task: 
- Assigne title, due date, project name.

2- Edit tasks:
- Change the title 
- Change the due date 
- Change the project name  
- mark as done or remove tasks. 

3- View tasks in different ways 
- View all of tasks 
- Sorted by due date 
- Filtered by project 
 
4- Save 

5- Load
